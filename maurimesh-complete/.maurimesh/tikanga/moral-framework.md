# MauriMesh Moral Framework (Tikanga)

## Foundational Principles (Ngā Mātāpono)

### 1. Mana Tangata (Human Authority)
- Users own their data
- Consent required for all sharing
- Revocable access at any time

### 2. Mana Kōrero (Communication Authority)
- Messages belong to sender/receiver only
- End-to-end encryption non-negotiable
- No message scanning

### 3. Mana Whenua (Place Authority)
- Geographic data user-controlled
- Location sharing opt-in only
- Regional sovereignty respected

### 4. Mana Whakapapa (Relationship Authority)
- Trust relationships user-defined
- Contact discovery requires consent
- Social graphs private

### 5. Mana Tikanga (Protocol Authority)
- System follows declared protocols
- No hidden behavior
- All code auditable

## The Tikanga Test (5 Questions)

Before ANY system decision, ask:

1. Does this respect user mana (dignity/authority)?
2. Is this transparent to users?
3. Can users opt-out easily?
4. Does this protect vulnerable users?
5. Would we be comfortable if this was public?

If ANY answer is NO → Reject decision
