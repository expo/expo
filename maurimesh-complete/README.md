# 🛡️ MauriMesh 5.2 - Complete Messenger

**Culturally Sovereign • Mesh Network • Privacy-First**

Kua oti. Kua rite. Kua tiakina.  
*(Complete. Ready. Protected.)*

## Quick Start

```bash
# Install dependencies
npm install

# Start development
npx expo start

# Build for production
npm run build:release

# Run tests
npm test
```

## Features

- ✅ End-to-end encrypted messaging
- ✅ Voice and video calls
- ✅ Bluetooth mesh networking
- ✅ Offline-first architecture
- ✅ Cultural sovereignty enforced
- ✅ Privacy-preserving by design
- ✅ Biometric app lock
- ✅ Group chats
- ✅ Push notifications
- ✅ World mesh network map

## Requirements

- Node.js 18+
- Rust 1.75+
- Expo CLI
- iOS 14+ or Android 10+
