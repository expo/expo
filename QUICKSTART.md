# 🚀 MauriMesh 5.2 Quick Start

## 1-Minute Setup

```bash
# Clone repository
git clone https://github.com/maurimesh/maurimesh-complete.git
cd maurimesh-complete

# Install dependencies
npm install

# Start development
npx expo start

# Scan QR code with Expo Go app
```

## First Use

1. **Create Identity** (automatic on first launch)
2. **Add Contacts** (QR code or phone number)
3. **Send Message** (tap contact → type → send)
4. **Explore Mesh** (Profile → Mesh Map)

## Build for Production

```bash
# Build release
npm run build:release

# Submit to stores
npm run submit:ios
npm run submit:android
```

## Run Tests

```bash
# Unit tests
npm test

# E2E tests
npm run test:e2e

# Coverage
npm run test:coverage
```

## Common Commands

| Command | Description |
|---------|-------------|
| `npm start` | Start dev server |
| `npm run lint` | Run linter |
| `npm run build` | Build app |
| `npm test` | Run tests |
| `npx expo start` | Start Expo |
