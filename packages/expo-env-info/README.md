## 🚀 Usage

### CLI

```sh
# Usage
npx expo-env-info [options] [path]
```

## ⚙️ Options

For more information run `npx expo-env-info --help` (or `-h`)

| Options       | Description               |
| ------------- | ------------------------- |
| -h, --help    | output usage information  |
| -v, --version | output the version number |
