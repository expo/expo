# MauriMesh 5.2 User Guide

## Getting Started

### Installation

1. Download MauriMesh from App Store or Google Play
2. Open the app
3. Create your identity (automatic)
4. Allow necessary permissions (Bluetooth, Notifications)

### Quick Start

1. **Add Contacts**
   - Scan QR code
   - Search by phone number
   - Discover nearby devices

2. **Send Messages**
   - Tap on a contact
   - Type your message
   - Tap send

3. **Make Calls**
   - Tap phone icon for voice call
   - Tap video icon for video call

## Troubleshooting

### Can't Connect
1. Check Bluetooth is enabled
2. Move closer to other users
3. Restart the app

### Messages Not Sending
1. Check internet connection
2. Messages will queue for mesh delivery
3. Wait for nearby devices

### Call Quality Issues
1. Check network connection
2. Reduce video quality
3. Switch to voice call
