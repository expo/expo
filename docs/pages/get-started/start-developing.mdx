import { useState } from 'react'; import { Card, CardContent } from "@/components/ui/card"; import { Input } from "@/components/ui/input"; import { Button } from "@/components/ui/button"; import { Tabs, TabsList, TabsTrigger, TabsContent } from "@/components/ui/tabs"; import { Label } from "@/components/ui/label"; import { Textarea } from "@/components/ui/textarea";

export default function SooqAlDawaaApp() { const [search, setSearch] = useState(""); const [newMed, setNewMed] = useState({ name: "", price: "", supplier: "" }); const [medicines, setMedicines] = useState([ { name: "باراسيتامول", price: "10 ج.م", supplier: "شركة الأمل" }, { name: "أموكسيسيلين", price: "25 ج.م", supplier: "فاركو" }, ]);

const filtered = medicines.filter(med => med.name.includes(search));

const handleAddMedicine = () => { if (newMed.name && newMed.price && newMed.supplier) { setMedicines([...medicines, newMed]); setNewMed({ name: "", price: "", supplier: "" }); } };

return ( <div className="p-4 max-w-md mx-auto rtl text-right"> <h1 className="text-2xl font-bold mb-4 text-center">سوق الدواء</h1>

<Tabs defaultValue="pharmacy" className="mb-4">
    <TabsList className="grid grid-cols-2">
      <TabsTrigger value="pharmacy">صيدلية</TabsTrigger>
      <TabsTrigger value="supplier">مورد</TabsTrigger>
    </TabsList>

    <TabsContent value="pharmacy">
      <Input
        placeholder="ابحث عن دواء..."
        value={search}
        onChange={e => setSearch(e.target.value)}
        className="mb-4"
      />
      <div className="space-y-2">
        {filtered.map((med, idx) => (
          <Card key={idx}>
            <CardContent className="p-4">
              <p className="font-semibold">{med.name}</p>
              <p>السعر: {med.price}</p>
              <p>المورد: {med.supplier}</p>
            </CardContent>
          </Card>
        ))}
      </div>
    </TabsContent>

    <TabsContent value="supplier">
      <Card>
        <CardContent className="p-4 space-y-4">
          <p className="font-bold text-lg">إضافة دواء جديد</p>
          <div>
            <Label>اسم الدواء</Label>
            <Input value={newMed.name} onChange={e => setNewMed({ ...newMed, name: e.target.value })} />
          </div>
          <div>
            <Label>السعر</Label>
            <Input value={newMed.price} onChange={e => setNewMed({ ...newMed, price: e.target.value })} />
          </div>
          <div>
            <Label>اسم المورد</Label>
            <Input value={newMed.supplier} onChange={e => setNewMed({ ...newMed, supplier: e.target.value })} />
          </div>
          <Button className="w-full" onClick={handleAddMedicine}>إضافة</Button>

          <hr className="my-4" />
          <p className="font-bold text-lg">الأدوية الحالية</p>
          <div className="space-y-2">
            {medicines.map((med, idx) => (
              <Card key={idx}>
                <CardContent className="p-4">
                  <p className="font-semibold">{med.name}</p>
                  <p>السعر: {med.price}</p>
                  <p>المورد: {med.supplier}</p>
                </CardContent>
              </Card>
            ))}
          </div>
        </CardContent>
      </Card>
    </TabsContent>
  </Tabs>
</div>

); }
