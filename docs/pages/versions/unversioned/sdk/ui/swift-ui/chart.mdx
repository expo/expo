Expo UI Chart wraps Apple's [Swift Charts](https://developer.apple.com/documentation/charts) framework, and supports rendering line, point, bar, area, pie, and rectangle charts with reference lines.

> **info** `Chart` requires iOS 16+ and tvOS 16+. Pie charts (`type="pie"`) additionally require iOS 17+ and tvOS 17+.

## Usage

### Line chart

```tsx LineChartExample.tsx
import { Host, Chart } from '@expo/ui/swift-ui';

export default function LineChartExample() {
  return (
    <Host style={{ flex: 1 }}>
      <Chart
        type="line"
        data={[
          { x: 'Jan', y: 15 },
          { x: 'Feb', y: 25 },
          { x: 'Mar', y: 18 },
          { x: 'Apr', y: 32 },
        ]}
        style={{ flex: 1 }}
      />
    </Host>
  );
}
```
