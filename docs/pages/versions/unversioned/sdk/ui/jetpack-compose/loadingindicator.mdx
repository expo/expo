## Usage

### Default loading indicator

<Tabs>
  <Tab label="Preview">
    <ContentSpotlight
      alt="LoadingIndicator component on Android."
      src="/static/images/expo-ui/loadingindicator/android.png"

  <Tab label="Code">
    ```tsx
    import { LoadingIndicator } from '@expo/ui/jetpack-compose';

    {/* Indeterminate (continuous animation) */}
    <LoadingIndicator />

    {/* Determinate with progress value */}
    <LoadingIndicator progress={0.5} color="#6200EE" />
    ```

  </Tab>
</Tabs>

### Contained loading indicator

A loading indicator inside a circular colored background.

```tsx
import { ContainedLoadingIndicator } from '@expo/ui/jetpack-compose';

<ContainedLoadingIndicator
  containerColor="#E8DEF8"
  color="#6200EE"
/>
```
