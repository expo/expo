`expo-age-range` provides access to user age range information. It uses Google's [Play Age Signals API](https://developer.android.com/google/play/age-signals/use-age-signals-api) on Android and Apple's [Declared Age Range framework](https://developer.apple.com/documentation/declaredagerange/) on iOS.

This API allows you to request age range information from users to help you comply with age-appropriate content regulations and provide age-appropriate experiences in your app.

Limitation: On Android, the Play Age Signals API (beta) throws an exception until January 1, 2026, [as documented](https://developer.android.com/google/play/age-signals/use-age-signals-api#integrate-play-age-signals). From January 1, the API will return live responses.

## Configuration in app config

### Setup iOS project

To use the age range API on iOS, you need to add the `com.apple.developer.declared-age-range` entitlement to your app. Add it to your **app.json** or **app.config.js**:

```json app.json
{
  "expo": {
    "ios": {
      "entitlements": {
        "com.apple.developer.declared-age-range": true
      }
    }
  }
}
```

<ConfigReactNative>

For bare React Native projects, add the entitlement to your **ios/[app]/[app].entitlements** file:

```xml
<key>com.apple.developer.declared-age-range</key>
<true/>
```

</ConfigReactNative>

## Usage

<SnackInline label="Age Range Usage" dependencies={['expo-age-range']} platforms={['android', 'ios', 'web']}>

import * as AgeRange from 'expo-age-range';

const result = await AgeRange.requestAgeRangeAsync({
  threshold1: 10,
  threshold2: 13,
  threshold3: 18,
});
