**package.json** is a JSON file that contains the metadata for a JavaScript project.

This is a reference to Expo specific properties that can be used in the **package.json** file.

## `install`

### `exclude`

Exclude a specific library from the version checks handled by `npx expo start` or `npx expo-doctor` by adding the `install.exclude` property to the **package.json** file.

```json package.json
{
  /* @hide Other package.json properties... */ /* @end */
  "expo": {
    "install": {
      "exclude": ["expo-updates", "expo-splash-screen"]
    }
  }
}
```

Any library listed in the `exclude` array will be ignored, but the [ `npx expo install` command](/more/expo-cli/#install) will install, check, and fix all other dependencies.

## `autolinking`

Allows configuring module resolution behavior by using `autolinking` property in **package.json**.

For complete reference, see [Autolinking configuration](/modules/autolinking/#configuration).
