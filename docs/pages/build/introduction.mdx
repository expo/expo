Yash Classes - Student Management App

A comprehensive mobile application for managing student attendance, classes, fees, and parent communications for Yash Classes.


Features Implemented

1. Class Management


Add Classes: Create new classes with names and descriptions

Delete Classes: Remove classes from the system

View Classes: Display all classes with quick access to student management

Class Navigation: Easy navigation between classes and their students


2. Student Management


Add Students: Add students to classes with details including:
Student name
Mobile number
Parent name
Parent mobile number
Email address

Delete Students: Remove students from classes

View Students: Display all students in a class

Student Details: Store and manage comprehensive student information


3. Attendance Tracking


Mark Attendance: Mark students as present or absent

Date Selection: Navigate through dates to mark attendance for any day

Attendance History: View attendance records for all students

Attendance Statistics: See overall attendance percentages and trends

Bulk Attendance: Mark attendance for entire classes at once

Filter Options: Filter attendance by status (Present/Absent/All)


4. Fee Management


Create Fees: Add fee records for students with:
Fee amount
Due date
Description
Status (Pending/Paid)

Fee Tracking: Monitor pending and paid fees

Mark as Paid: Update fee status when payment is received

Fee Statistics: View total pending fees and payment status

Fee History: Track all fee transactions


5. Parent Communication


WhatsApp Messaging: Send bulk WhatsApp messages to all parents in a class

SMS Notifications: Send SMS messages for important updates

Message Templates: Pre-built templates for:
Holiday announcements
Time change notices
Fee reminders

Bulk Messaging: Send messages to all parents with one click

Message History: Track all sent messages with timestamps and recipient count


6. Dashboard & Home Screen


Statistics Overview: Display key metrics:
Number of classes
Total students
Average attendance percentage

Quick Actions: Fast access to:
Mark attendance
View history
Send messages

Navigation: Easy access to all features via tab navigation


Technology Stack

Frontend


React Native: Cross-platform mobile development

Expo: Managed React Native platform

TypeScript: Type-safe development

TailwindCSS (NativeWind): Utility-first styling

Expo Router: File-based routing


Backend


Node.js: Server runtime

tRPC: Type-safe API framework

Drizzle ORM: Database ORM

MySQL/TiDB: Database


Authentication


Manus OAuth: Secure user authentication


Database Schema

Tables Created


users: User authentication and profile

classes: Class information

students: Student details with parent contact info

attendance: Attendance records with date and status

fees: Fee records with payment status

messages: Message history for tracking communications

API Endpoints (tRPC)

Classes


classes.list - Get all classes

classes.create - Create a new class

classes.update - Update class details

classes.delete - Delete a class


Students


students.listByClass - Get students in a class

students.create - Add a student

students.update - Update student details

students.delete - Remove a student


Attendance


attendance.markAttendance - Mark attendance for a student

attendance.getByClassAndDate - Get attendance for a class on a specific date


Fees


fees.listByStudent - Get fees for a student

fees.listByClass - Get fees for a class

fees.create - Create a fee record

fees.markAsPaid - Mark fee as paid


Messages


messages.sendBulkMessage - Send message to all parents

messages.getHistory - Get message history

Getting Started

Prerequisites


Node.js 18+

pnpm package manager

Expo CLI


Installation

# Install dependencies
pnpm install

# Set up database
pnpm db:push

# Start development server
pnpm run dev

Running on Mobile

# For iOS
pnpm run ios

# For Android
pnpm run android

# For Web
pnpm run web

Usage Guide

Creating a Class


Navigate to the "Classes" tab

Click "Add Class"

Enter class name and description

Click "Create"


Adding Students


Go to Classes tab

Select a class

Click "Students"

Click "Add Student"

Enter student and parent details

Click "Add Student"


Marking Attendance


Go to "Attendance" tab

Select the date

Choose "Present" or "Absent"

Click "Save Attendance"


Managing Fees


Go to "Fees" tab

Click "Add Fee" to create new fee record

Enter amount and description

Click "Mark as Paid" when payment is received


Sending Messages


Go to "Messages" tab

Click "Send Message"

Select message type (WhatsApp/SMS)

Choose template or write custom message

Click "Send to All Parents"
