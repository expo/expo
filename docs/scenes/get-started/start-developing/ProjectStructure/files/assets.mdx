<RawH3>assets</RawH3>

The **assets** directory contains an **expo.icon** directory for the iOS app icon and an **images** subdirectory with Android adaptive icon files (**android-icon-foreground.png**, **android-icon-background.png**, **android-icon-monochrome.png**), a **splash-icon.png** for the splash screen, a **favicon.png** for the web, and tab navigation icons.
